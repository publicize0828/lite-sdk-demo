# 魔珐数字人能力展示 Demo

基于魔珐有灵（XmovAvatar）SDK 的数字人能力展示应用，集成 AI 智能助手、实时语音交互、SSML 动作控制、Widget 卡片展示等功能。

## 体验地址

👉 [https://publicize0828.github.io/lite-sdk-demo/](https://publicize0828.github.io/lite-sdk-demo/)

## 功能

- 🎭 数字人渲染：多角色、音色、背景切换
- 🤖 AI 智能助手：DeepSeek V4 Pro 驱动，流式对话
- 🎤 语音交互：ASR 语音识别 + LLM 对话
- 🏃 SSML 动作：行走、指向、意图动作、Widget 卡片
- 📝 代码游乐场：在线 HTML 编辑预览
- 📚 互动教程：Step by step SDK 接入

## 本地开发

```bash
pnpm install
pnpm dev
```
